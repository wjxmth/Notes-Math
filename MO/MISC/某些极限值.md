# 某些极限值



三角函数
$$
\lim_{x \to 0} \frac{\sin x}{x} = 1
$$


对数
$$
\lim_{x \to 0^+} \frac{\ln x}{x} = 0
$$

> 证明：先证明当 $0 < x < \dfrac{1}{4}$ 时有 $\ln x > - \dfrac{1}{\sqrt{x}}$，所以有 $x\ln x > x \left(- \dfrac{1}{\sqrt{x}} \right) = - \sqrt{x}$，又因为 $\lim_{x\to 0^+} (-\sqrt{x}) = \lim_{x\to 0^+} 0 = 0$，然后用夹逼定理即可证明.
